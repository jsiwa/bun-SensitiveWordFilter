# bun-word-filter

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

## wrk
```bash
wrk -t4 -c10 -d30s http://localhost:8080 -s wrk-post.lua
```

This project was created using `bun init` in bun v1.1.7. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
